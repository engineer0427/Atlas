# Atlas – Research Network Framework

Atlas는 논문, 코드, 데이터셋을 연결해 연구 네트워크를 **지도처럼 탐색**할 수 있는 파이썬 프레임워크입니다.


## 설치
```bash
pip install -e .
```

## 결과
- `example01.html, example02.html` → 브라우저에서 열기
